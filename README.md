====

Google翻訳の結果をロンダリングする Gem です。

## Description

Googleの翻訳結果を複数適用したテキストを返します。

## Demo

```
bin/translaunder ja en 事なきを得る
=> 私はKotonakiを取得
```

## Install

```
gem install translaunder
```
